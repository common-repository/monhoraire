Proposez dès maintenant la prise de rendez-vous en ligne sur votre site internet Wordpress avec MonHoraire.

== Description ==

MonHoraire est un service d’agenda et de marketing en ligne.

Les fonctionnalités incluent un agenda multi-utilisateurs qui est à la fois ergonomique et simple d'utilisation, la prise de rendez-vous en ligne et le rappel des rendez-vous par SMS.

Avec MonHoraire, fidélisez et engagez vos clients en leur simplifiant la prise de rendez-vous.

== Installation ==

= Depuis votre tableau de bord WordPress =

1. Allez dans 'Extensions > Ajouter'
2. Rechercher l'extension 'MonHoraire'
3. Une fois trouvé, cliquez sur 'Installer'
4. Une fois installé, allez dans 'Extensions' et activer l'extension MonHoraire

= Depuis WordPress.org =

1. Allez sur le site 'https://wordpress.org/plugins/' et rechercher 'MonHoraire'
2. Cliquez dessus et téléchargez-le en cliquant sur le bouton 'Télécharger'
3. Une fois télécharger, déplacer le dossier télécharger dans le dossier '/wp-content/plugins', avec la méthode de votre choix (ftp, sftp, ...)
4. Allez dans 'Plugin' depusi votre tableau de bord Wordpress et activer votre nouvelle extension

== Frequently Asked Question ==

= A quoi correspondent tous ces champs dans la configuration de l'extension? =

Vous avez cinq paramètres pour configurer l'extension selon vos besoins :
1. Titre : Ce champ contient le titre qui apparaîtra comme le titre de l'extension sur votre site web. Vous pouvez choisir un titre comme 'Prenez rendez-vous' ou encore le nom de votre entreprise.
2. Description : Vous pouvez mettre une description sous le titre de l'extension. Une courte description de votre activité est conseillé.
3. URL : Ce champ est important. C'est lui qui va gérer l'affichage du module de prise de rendez-vous.
4. Largeur : Ce champ correspond à la largeur de l'extension dans votre site.
5. Hauteur : Ce champ correspond à la hauteur de l'extension dans votre site.

= Que mettre exactement dans le champ URL ? =

Pour connaître votre adresse URL, vous pouvez allez sur 'https://www.monhoraire.pro', vous connectez et copier l'adresse qui apparaît dans la barre URL de votre navigateur. Il ne vous reste qu'à copier le lien dans le champ URL.

= Pourquoi je ne peux pas de valeur nulle dans les champs 'Largeur' et 'Hauteur'? =

Pour être sûr que l'extension apparaîsse sur votre site, les valeurs par défauts sont 500. Mais vous pouvez très bien modifier ces valeurs pour adapter l'extension à votre site.

= Je veux mettre une très grande taille dans le champ 'Hauteur' ou 'Largeur' mais les dimensions de l'extension sur mon site ne changent pas. Pourquoi? =

Si vous avez placé l'extension dans une 'sidebar', par exemple, les dimensions de l'extension ne peuvent être plus grandes que celles de la 'sidebar'. Pour palier ce problème, vous pouvez mettre l'extension dans une page qui lui sera entièrement dédié.
